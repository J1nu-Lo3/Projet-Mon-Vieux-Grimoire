# Mon vieux Grimoire

## Comment lancer le projet ?

### Avec npm

Faites la commande `npm install` pour installer les dépendances puis `npm start` pour lancer le projet accessible sur `http://localhost:3000`.

Le projet a été testé sur node 19.

### Avec npm dans backend

Placez-vous danc le backend `cd backend` puis faites la commande `npm install` pour installer les dépendances,
puis `npm start` pour lancer le serveur normalement
ou `npx nodemon` .

### Technologies utilisées Backend

Node.js
Express
MongoDB / Mongoose
Sharp
JWT (jsonwebtoken)
Bcrypt (bcryptjs)
Multer
Nodemon
